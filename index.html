Recent & Active Projects

  • Pyret is a functional scripting language designed for programming education.
  • Bootstrap builds and does professional development for curricula that integrate computing into middle- and high-school courses. Bootstrap 2 and Bootstrap Physics use Pyret.
  • In-flow Peer Review integrates code review into programming assignments. I've studied the effects of in-flow peer review of tests.
