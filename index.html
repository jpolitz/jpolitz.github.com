Systems & Software

  • code.pyret.org is the best way to run Pyret programs.
  • Pyret is a functional scripting language designed for programming education.
  • Macaroons are better cookies, and have several third-party implementations and extensions. They grew out of the Google Belay project, which explored secure sharing and identity management in the cloud.
  • λπ is an experimental semantics for Python. It represents a global effort, read about how we built it here.
  • S5 is a JavaScript implementation designed for exploration. This video shows a bit of S5 in action. With Matt Carroll, Ben Lerner, and Justin Pombrio.
    (hide)
  • I designed a novel type-checking strategy for the verification of reference monitors. I used it to mechanically verify ADsafe, a web sandboxing framework, with Arjun Guha, Spiridon Eliopoulos, and Shriram Krishnamurthi.
