Systems & Software

  • (New!) λπ is an experimental semantics for Python. It represents a global effort, read about how we built it here.
  • S5 is a JavaScript implementation designed for exploration. This video shows a bit of S5 in action. With Matt Carroll, Ben Lerner, and Justin Pombrio.
    (hide)
  • Google Belay explored secure sharing and identity management in the cloud. The code for our demo is open source and available. With Arjun Guha, Mark Lentczner, and Úlfar Erlingsson.
    (hide)
  • I designed a novel type-checking strategy for the verification of reference monitors. I used it to mechanically verify ADsafe, a web sandboxing framework, with Arjun Guha, Spiridon Eliopoulos, and Shriram Krishnamurthi.
  • Assistment is tutoring and assessment software for middle school math students, and part of learning sciences research at WPI CS with Neil Heffernan and many others.
