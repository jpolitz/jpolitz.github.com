Recent & Active Projects

  • code.pyret.org is the best way to run Pyret programs.
  • Pyret is a functional scripting language designed for programming education.
  • Bootstrap builds and does professional development for curricula that integrate computing into middle- and high-school courses. The Reactive, Data Science, and Physics curricula use Pyret.
  • Stopify is a JavaScript compiler and runtime for execution control (debugging, pausing, stopping) of JavaScript programs in stock web browsers.
